# Test Personas

## Sarah - First-Time Visitor
- Age: 28
- Tech comfort: Moderate
- Goal: Understand what this product does and whether to sign up
- Behavior: Scans headings, looks at pricing, reads 1-2 testimonials, leaves if confused within 30 seconds
- Patience: Low
- Will fill forms: Only if value is clear and fields are minimal

## Marcus - Returning Power User
- Age: 35
- Tech comfort: High
- Goal: Complete a specific task as fast as possible
- Behavior: Ignores onboarding, goes straight to features, uses keyboard shortcuts, expects search to work
- Patience: Very low for things that slow him down
- Will fill forms: Only required fields, auto-fill everything possible

## Elena - Accessibility-Dependent User
- Age: 45
- Tech comfort: Moderate
- Goal: Same tasks as any user, using screen reader and keyboard only
- Behavior: Navigates by headings, relies on link text, needs ARIA landmarks
- Patience: High for necessary complexity, zero for inaccessible interfaces
- Will fill forms: Needs proper labels and error association

## David - Non-Technical Executive
- Age: 52
- Tech comfort: Low
- Goal: Find specific information, usually under time pressure
- Behavior: Clicks the biggest button, reads minimal text, calls support if stuck
- Patience: Extremely low
- Will fill forms: Abandons if more than 5 fields

## Aisha - Mobile-Only User
- Age: 22
- Tech comfort: High (mobile), Low (desktop)
- Goal: Quick interactions while multitasking
- Behavior: Thumb-zone navigation, expects swipe gestures, frustrated by non-mobile-optimized sites
- Patience: Low, will switch to competitor app
- Will fill forms: Only with auto-fill, hates typing on mobile

## Tom - Skeptical Comparison Shopper
- Age: 40
- Tech comfort: Moderate
- Goal: Evaluate this against 3 competitors
- Behavior: Looks for pricing, feature lists, reviews, tries free tier, checks refund policy
- Patience: Medium (willing to explore but not to struggle)
- Will fill forms: Not for free trial if credit card required
